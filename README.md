# sf-async
## Requirements
Ensure the following tools are installed on your machine:
- [Docker](https://docs.docker.com/get-docker/) : Used to build and run the application containers
- [Docker Compose](https://docs.docker.com/compose/): Used to orchestrate the application services
- [GNU Make](https://www.gnu.org/software/make/): Used to simplify common development tasks.
- [Git](https://git-scm.com/downloads): Versioning tools

Verify your installation:

```bash
docker --version
docker compose version
make --version
git --version
```

## Installation
1. Clone the repository on your local machine
2. launch the command `make install`
3. Start the project with `make start`
4. The project is available on [http://localhost:8881/](http://localhost:8881/)

For more details and contributions, see more [here](docs/index.md)
